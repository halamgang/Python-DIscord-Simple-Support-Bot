# 🤖 Python Discord Support Bot
> 파이썬 기반의 직관적이고 가벼운 디스코드 고객 지원(Support) 봇입니다.

## 📝 프로젝트 소개
사용자가 문의를 남기면 관리자에게 연결해 주는 티켓형 지원 시스템입니다. 
단순한 코드 구성을 넘어 실제 클라우드 환경에 배포하여 운영한 경험을 담고 있습니다.

## 🌐 호스팅 및 배포 (Hosting)
- **Platform:** [Cloudtype (클라우드타입)](https://cloudtype.io/)
- **Status:** 클라우드 환경에서 24/7 상시 가동되도록 호스팅 설정을 완료하였습니다.

## 🛠 기술 스택
- **Language:** Python 3.10+
- **Library:** discord.py
- **Platform:** Cloudtype PaaS\

## 개발 사유
- 당시 discord의 bot에 관심이 많았어 고등학교 시절 구현해봤습니다.
